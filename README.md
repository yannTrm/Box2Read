# ocr
Custom OCR framework , extract and interpret data from vehicle-related documents and identifiers. This repository specializes in reading license plates, VIN, odometers, and registration documents .


# Idea of Network


2 steps models :
- CRNN
- MobileViT + RNN
- ViTSTR

end to end models:
- Donut
- TrOCR


# How to train

vanilla training
transfer learning
